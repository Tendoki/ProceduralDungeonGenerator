# ProceduralGeneratorDungeon
## Процедурный генератор подземелий в 2D
В сцене лежит три объекта, в каждом из которых есть скрипт, который генерирует разные объекты в 2D пространстве:
* Объект "RandomWalkRoomGenerator" генерирует случайным блужданием агента комнату с указанными параметрами
* Объект "RandomWalkDungeonGenerator" генерирует с помощью случайного блуждания агента подземелье с указанными параметрами
* Объект "BSPDungeonGenerator" генерирует с помощью бинарного разделения пространства подземелье с указанными параметрами
